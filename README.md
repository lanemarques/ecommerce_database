## Projeto Banco de Dados - E-commerce 
📌 Descrição 

Este projeto consiste na modelagem e implementação de um banco de dados para um sistema de e-commerce, utilizando MySQL. 

O objetivo é representar clientes, pedidos, produtos, fornecedores, vendedores, pagamentos e entregas, além de realizar consultas que simulem situações reais de negócio. 

🧱 Estrutura do Projeto O projeto está dividido em dois arquivos principais: 

- BD-ecommerce.sql
   - Contém a criação do banco de dados e das tabelas
-
- inserts_consultas.sql
   - Contém inserção de dados e consultas SQL e refinamento 
   -
🗃️ Modelagem
     O banco de dados inclui as seguintes entidades:
  - Clientes (PF e PJ)
  - Produtos
  - Pedidos
  - Pagamentos
  - Entregas
  - Fornecedores
  - Vendedores
  - Estoque

  🔍 Consultas Implementadas:
   Foram realizadas consultas utilizando:
  SELECT
  WHERE
  JOIN
  ORDER BY
  GROUP BY
  HAVING
  Funções agregadas (SUM, COUNT)

  Exemplos:
  Produtos mais vendidos
  Pedidos por cliente
  Tipo de pagamento mais utilizado
  Produtos por fornecedor
  Vendedor que mais vendeu
  Produtos entregues
  Pagamentos em dinheiro

⚙️ Tecnologias MySQL 

▶️ Como executar 
Criar o banco de dados 
Executar o arquivo BD-ecommerce.sql onde constam as tabelas 
Executar o arquivo queries_data_insertion.sql onde constam as inserçoes e consultas
